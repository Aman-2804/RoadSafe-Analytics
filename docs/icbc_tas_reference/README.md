# ICBC TAS (Traffic Accident System) Reference Data

This directory contains ICBC TAS reference files used for data mapping and validation.

## Files

- `tas_e02_metadata.csv` - TAS E02 metadata definitions
- `tas_e02_codes.csv` - TAS E02 code mappings and descriptions

## Purpose

These reference files enable:
- **Data Mapping**: Translate raw data codes to meaningful descriptions
- **Data Validation**: Verify that codes in source data match ICBC standards
- **Business Context**: Understand ICBC-specific classifications and terminology

## Usage

These files are referenced during:
- ETL transformation processes
- Data quality validation
- Business rule enforcement
- Report generation

## ICBC TAS Overview

The Traffic Accident System (TAS) is ICBC's standardized system for recording and classifying traffic collisions. E02 refers to a specific data element or classification within the TAS schema.
